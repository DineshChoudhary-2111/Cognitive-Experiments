# Sleep and Cognitive Tracking System

## Objective
To explore how sleep habits, screen time, mood, and focus levels influence cognitive performance including reaction time and working memory.

## Variables Tracked
- Sleep duration
- Sleep quality
- Sleep and wake timing
- Screen time before sleep
- Time to fall asleep
- Mood
- Focus levels
- Reaction time
- Working memory performance

## Methodology
Data was collected manually over a period of 10 days from May 2 to May 11.  
Participants recorded daily sleep-related variables using a Python-based logging system.  
Cognitive performance was measured using separate reaction time and working memory tasks.  
All datasets were later merged for correlation analysis between sleep habits and cognitive performance.

## Technologies Used
- Python
- Pandas
- Matplotlib
- CSV data logging

## Observations
Initial patterns suggest that better sleep quality and higher sleep duration are associated with improved reaction time and higher memory performance.  
Higher screen time and longer sleep latency show weaker cognitive performance trends in some cases.

## Future Scope
- Larger dataset collection over extended periods
- More participants for stronger statistical validity
- Advanced correlation and regression modeling
- Sleep stage or wearable integration
