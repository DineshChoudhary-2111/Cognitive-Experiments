# Sleep and Cognitive Tracking System

## Objective
To explore possible relationships between sleep habits, screen time, mood, and cognitive performance.

## Variables Tracked
- Sleep duration
- Wake/sleep timing
- Screen time
- Mood
- Focus levels
- Reaction time
- Memory performance

## Methodology
Daily lifestyle and sleep variables were recorded and compared against cognitive task performance over multiple days.

## Technologies Used
- Python
- CSV data logging

## Observations
Patterns suggested possible correlations between sleep quality, reaction time, focus, and memory consistency.

## Future Scope
Longer-term tracking and statistical correlation analysis across larger datasets.
