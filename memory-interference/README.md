# Memory Interference Experiment

## Objective
To study how cognitive interference affects short-term memory recall and response accuracy.

## Variables Measured
- Memory accuracy
- Interference task accuracy
- Recall response time

## Methodology
Participants memorized a 4-number sequence.  
Interference trials introduced timed arithmetic tasks before recall.  
Control trials involved direct recall without interference.

## Technologies Used
- Python
- CSV data logging

## Observations
Interference conditions generally reduced recall accuracy and increased response time.

## Future Scope
Larger participant groups and statistical analysis across cognitive load conditions.
