# Memory Interference Experiment

## Objective
To investigate how cognitive interference affects short term memory retention recall accuracy and retrieval speed under divided attention conditions.

## Variables Measured
- Memory recall accuracy
- Recall response time
- Interference task accuracy

## Methodology
Participants memorized randomized 4 number sequences under two experimental conditions.

In control trials participants directly recalled sequences after memorization.

In interference trials participants first completed timed arithmetic tasks before recall. These secondary tasks were designed to introduce cognitive load and disrupt active rehearsal processes within working memory.

Trials were randomized to reduce adaptation effects across sessions.

## Cognitive Focus
This experiment primarily explored:
- working memory
- interference effects
- attentional division
- short term recall stability under cognitive load

The design was influenced by the idea that memory performance is shaped not only by storage capacity but also by competing mental activity occurring during retention periods.

## Technologies Used
- Python
- CSV based data logging
- Randomized trial generation

## Observations
Interference conditions generally produced reductions in recall accuracy and slight increases in retrieval variability across participants.

Results suggested that introducing secondary cognitive tasks can disrupt short term retention by competing for limited working memory resources.

## Future Scope
Possible future extensions include:
- larger participant groups
- adaptive interference difficulty
- variable sequence lengths
- multimodal interference conditions
- statistical analysis of cognitive load patterns
